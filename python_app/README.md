# Typhoon Deep Research Python

This directory contains a minimal FastAPI reimplementation of the
Typhoon Deep Research backend. It mirrors the structure of the
original Next.js project's API routes but uses Python.

## Development

```bash
# Install dependencies
pip install -e .

# Run the server
uvicorn app.main:app --reload
```

The endpoints are placeholders and do not perform real research yet.
They demonstrate how the Next.js logic could be ported to Python.
